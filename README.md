# Analisis-Statistik-Kemiskinan-Jawa-Timur-2024
Project ini membahas analisis data kemiskinan kabupaten/kota di Jawa Timur tahun 2024 menggunakan data dari Badan Pusat Statistik (BPS). Analisis dilakukan untuk mengetahui jumlah penduduk miskin, persentase kemiskinan, rata-rata, proporsi, serta perbedaan tingkat kemiskinan antar daerah.
